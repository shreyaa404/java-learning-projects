# Guess Number Game

## Description

A simple Java console-based game where the user has 5 attempts to guess a randomly generated number between 1 and 100.

## Concepts Used

* Variables
* Data Types
* Operators
* If-Else Statements
* For Loop
* Scanner Class
* Math.random()
* Break Statement

## Features

* Generates a random number between 1 and 100
* Gives the player 5 attempts
* Provides hints whether the guess is too high or too low
* Validates the input range
* Displays a win or lose message

## What I Learned

* Taking user input using Scanner
* Generating random numbers
* Building program logic using conditions
* Using loops for repeated attempts
* Handling edge cases in a simple game

## Future Improvements

* Handle non-numeric input
* Add difficulty levels
* Allow the user to play multiple rounds
* Track the number of attempts taken to win
