# Tic Tac Toe Game

## Learning Stage

This project was built after learning Java Methods and Arrays. It combines previously learned concepts such as loops, conditional statements, methods, arrays, and user input to create a complete console-based game.

## Description

A two-player Tic Tac Toe game developed in Java. Players take turns placing X and O on a 3×3 board. The game checks for winning conditions and draw situations after every move.

## Concepts Practiced

* Methods
* Arrays (2D Array)
* Loops
* Conditional Statements
* Scanner Class
* Basic Problem Solving

## What I Learned

* Managing game state using arrays
* Converting user input into board positions
* Breaking a large problem into methods
* Implementing game logic step by step
* Validating user input
